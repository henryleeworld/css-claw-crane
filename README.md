# CSS 娃娃機

一種電子遊戲機，它將獎勵物品陳列於透明箱內，當玩家投入錢幣後，可以控制一個機械手臂來抓取其中的物品，如果成功抓住並被自動放入取物孔則可獲得該物品作為獎品。有些抓物機會設定保證價格，也就是玩家投幣滿機台設定的金額後，可以一直嘗試夾取物品直到成功到手。

## 使用方式
- 把整個專案複製一份到你的電腦裡，這裡指的「內容」不是只有檔案，而是指所有整個專案的歷史紀錄、分支、標籤等內容都會複製一份下來。
```sh
$ git clone
```
